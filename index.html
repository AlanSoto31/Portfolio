Tic-Tac-Toe Game

Ruby Rspec

Tic-tac-toe is a paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3×3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row is the winner. I created the game using Ruby, it has a basic interface that runs in the terminal.The goals of this project are learn how to create and structure a Ruby project and learn to manage the files with correct usage to make code maintainable.
